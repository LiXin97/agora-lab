# Reviewer Independence

## Core Rule

Reviewers must judge primary artifacts directly rather than relying on someone else's summary.

## Allowed Inputs

- file paths
- review objective
- rubric
- venue constraints

## Disallowed Inputs

- pre-digested summaries
- leading conclusions
- suggested verdicts
- filtered evidence that hides weaknesses

## Why

Independent review only works when the reviewer forms its own judgment from raw artifacts.
